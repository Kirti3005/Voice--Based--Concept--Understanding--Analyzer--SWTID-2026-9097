# VBCUA

VBCUA is an AI-powered cybersecurity application that uses Machine Learning and Generative AI to detect threats, analyze anomalies, and provide security insights through an interactive Streamlit interface.

## Features

- Threat detection using Machine Learning
- Anomaly analysis
- AI-generated security insights
- Interactive dashboard
- Data visualization
- Dataset upload and analysis

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Plotly

## Installation

```bash
git clone https://github.com/your-username/VBCUA.git
cd VBCUA
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run app.py
```

The application will be available at:

```
http://localhost:8501
```
